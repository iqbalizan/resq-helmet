# ResQ Helmet

Smart helmet safety system developed using Raspberry Pi Pico W and CircuitPython.

## Project Overview

ResQ Helmet is a smart safety helmet designed to improve rider safety through real-time monitoring and emergency alert features.

## Features

- Crash detection using MPU6050
- Telegram SOS notification
- Ambient light monitoring
- Buzzer warning system
- Real-time sensor monitoring
- Limit switch for helmet usage detection

## Hardware Used

- Raspberry Pi Pico W
- MPU6050
- LDR Sensor
- Buzzer
- Limit Switch

## Software Used

- CircuitPython
- Telegram Bot API
